Websiten erstellen und Gestalten

Willkommen

Websiten bestehen aus recheckigen Kästchen. Alles Runde ist entweder Trick, Grafik oder beides. Beim Umgang mit diesen Kästchen haben HTML und CSS klar getrennte Aufgaben:

  • HTML strukturiert die Website und erstellt Kästchen
  • CSS gestaltet die Kästchen und die Inhalte darin

Beide Sprachen arbeiten bei der Erstellung und Gestaltung von Websiten eng zusammen.
